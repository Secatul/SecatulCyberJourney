## Microsoft 2019
Em dezembro de 2019, a Microsoft enfrentou uma **violação massiva de dados** envolvendo seu banco de dados interno de suporte ao cliente.

**250 milhões de registros** foram expostos na internet

Tipo de dados vazados: Logs completos de interações entre clientes Microsoft
- Endereços de email dos clientes
- Endereços IP
- Localizações geográficas
- Histórico completo das conversas

#### **Como Foi Descoberta?**
Dois mecanimsos de busca (BinaryEdge e Comparitech) indexaram essas bases de dado.
