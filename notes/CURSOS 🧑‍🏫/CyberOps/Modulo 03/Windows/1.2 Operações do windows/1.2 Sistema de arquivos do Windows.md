### exFAT (Extenisble File Allocation Table)
Sistema de arquivos optimizado para USBs flash (Pendrives), criado para substituir o FAT32 pois permite armazenar arquivos com mais de 4 GB.

Saiba mais: https://en.wikipedia.org/wiki/ExFAT
### HFS+
Sistema de arquivos principal da Apple, porém foi substituido pelo APFS.

Saiba mais: https://en.wikipedia.org/wiki/HFS_Plus
### EXT
Extended File System, é o sistema de arquivos do Linux

Saiba mais: https://en.wikipedia.org/wiki/Extended_file_system

### NTFS
NT File System, sistema de arquivos mais comumente usado ao instalar o Windows.

Tópicos mais pra frente, entender ADS (Alternate data stream) possível lugar para esconder malware.
