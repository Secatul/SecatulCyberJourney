Abrindo o terminal podemos testar alguns comandos

```bash
dir --> # mostra diretórios (mesma coisa do ls no linux)
cd / --> # diretório raiz
cd .. --> # muda para uma pasta acima do diretório atual
copy [destino] --> # copia arquivos para outro local
del [nome do arquivo] --> # exclui arquivo
find --> # procura texto em arquivos (help find) para mais informações
ren [nome_antigo] [nome_novo]
```
